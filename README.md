

# Agentarium

Browser-native demos for inspectable AI agent systems.

## Packages

```txt
packages/
  core/          # Headless agent runtime primitives
  open-harness/  # Browser UI for inspecting agent execution
```

## OpenHarness

OpenHarness is a browser-only agent harness that exposes the runtime state of a lightweight local AI agent.

It shows:

- prompt and context assembly
- tool calls and tool results
- memory and context updates
- retries and failures
- trace events
